Parse is all about making life easier for developers.With that in mind, we’ve been hard at work adding new Cloud Modules to take care of some of the most common tasks users want to perform in Cloud Code.Today we’re launching two new modules: the Parse Image Cloud Module, and the Parse User Session Module.Parse Image Cloud Module With the Parse Image Cloud Module, you can manipulate graphical images such as photos easily.For example, it’s easy to create a Cloud Code Function to make a thumbnail of a photo.

We’re going to take you through a case where one requires its website visitors to upload an image to the server.It’s quite easy to handle this in a rails application but you will have to create Amazon S3 or Dropbox account for file upload as most of the Rails PaaS providers don’t support file system.Now we will proceed with a simple and small Rails project using App42 File Upload APIs.In App42, the uploaded file goes to CDN network and gives faster access time to your app users, and can be accessed from http endpoint.To get you started, let’s create a new Rails application called rails_image_upload.

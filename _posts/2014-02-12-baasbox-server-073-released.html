In this new release we have added a lot of new features to the server and fixed many issues.Furthermore, we have rewritten the Android SDK from scratch and improved the iOS SDK.New set of APIs to manage files (upload, metadata and content extraction), query on them; Added a new API that allows admins to reset users’ password; Added button to reset users’ password; Signup, Login and Profile (GET /me) APIs now return the same data; The APIs GET/PUT /user are deprecated, use /me instead;

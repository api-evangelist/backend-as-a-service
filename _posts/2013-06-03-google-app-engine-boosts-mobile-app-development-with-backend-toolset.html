Google appears to be getting more focused on bringing over mobile developers to its App Engine Platform-as-a-Service infrastructure with a new introduction on Monday.The Internet giant is adding on a Mobile Backend Starter, touted to be a cloud-based mobile backend platform with a general purpose client-side framework for Android.Brad Abrams, a product manager for the cloud developer program at Google, described further in a blog post on Monday that the system is a one-click deployable mobile backend with none of the headaches.Mobile Backend Starter gives you everything you need to rapidly set up a backend for your app, without needing to write any backend code.It includes a server that stores your data with App Engine and a client library and sample app for Android that make it easy to access that data.

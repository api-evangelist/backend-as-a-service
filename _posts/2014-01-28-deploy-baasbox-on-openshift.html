Download and decompress the source code of BaasBox from the GitHub repository.This is necessary because we need two script files: start_openshift and stop_openshift The GitHub repository can be found at this address https://github.com/baasbox/baasbox Install the necessary software needed by OpenShift.You can find the guide related to your system at https://www.openshift.

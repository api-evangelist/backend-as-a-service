HTML5 programs consist of three main components: the HTML text markup language for page layout, CSS3 for style, and JavaScript for in-browser execution of program code.And although it is possible to sit down at a text editor and just write your code, there are tools that streamline the process considerably.JQuery Mobile is one tool that provides extra help with pre-built widgets and CSS auto code generation.If you’re interested in learning more about developing HTML5 apps, check out a free tutorial in our free eBook.JQuery Mobile (jQM) is a “touch optimized” framework, meaning that it offers special widgets that streamline the writing of features (like buttons) important on a mobile phone.

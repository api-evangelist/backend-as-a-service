If you’re currently using our iOS SDK, we’d like to invite you to join our iOS SDK Beta program.We’re always striving to build the best and most stable SDKs and you can help!As part of the program, beta testers will: Test out the latest and greatest StackMob features before they’re officially announced.Provide feedback and file bug reports with our iOS SDK team on features in development.Be encouraged to fork our repository, contribute changes and merge code (upon approval from the iOS SDK team).
